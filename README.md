# Friendly - 设计友好报

React应用展示创意设计作品，包含交互式Three.js球体主页和三个内容页面。

## 功能特色
- 🌐 Three.js 交互式球体主页
- 🎨 Inspiration页面 - 带动画筛选器
- 👨‍🎨 Artist页面 - 艺术家作品展示  
- 🏢 Team Project页面 - 团队项目展示
- 📱 响应式设计
- ☁️ Supabase云存储集成

## 最新更新 (2024-07-09)

## 技术栈
- React 18
- Three.js
- Tailwind CSS
- Supabase
- GitHub Pages
